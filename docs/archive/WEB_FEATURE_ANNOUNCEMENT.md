# 🎉 新功能发布：Web 界面

## 📢 公告

我们很高兴地宣布，剧本叙事结构分析系统现已支持 **Web 界面**！

现在你可以通过浏览器轻松上传剧本、实时查看分析进度，并在美观的界面中查看详细结果。

---

## 🌟 主要特性

### 1. 📤 简单易用的文件上传
- 拖拽或点击上传 JSON 格式剧本
- 支持多种 LLM 提供商（DeepSeek、Anthropic、OpenAI）
- 文件验证和错误提示

### 2. ⏱️ 实时进度追踪
- WebSocket 实时推送分析进度
- 可视化进度条（10% → 40% → 70% → 90% → 100%）
- 三阶段流程状态展示

### 3. 📊 丰富的结果展示
**5 个交互式标签页：**

- **概览** - 统计摘要和性能指标
- **TCC 详情** - 每个戏剧冲突链的完整信息
- **线级排序** - A/B/C 线分级和评分
- **结构修正** - 发现的问题和修复建议
- **可视化** - Mermaid 流程图

### 4. 📥 一键下载报告
- 完整的 Markdown 格式分析报告
- 包含 Mermaid 流程图
- 可直接在 Markdown 编辑器中查看

---

## 🚀 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
pip install -r requirements-web.txt
```

### 启动服务器
```bash
./run_web_server.sh
```

### 访问界面
在浏览器中打开：**http://localhost:8000**

## 🛠️ 技术亮点

### 现代化技术栈
- **FastAPI** - 高性能异步 Web 框架
- **WebSocket** - 实时双向通信
- **Bootstrap 5** - 响应式 UI 设计
- **Mermaid.js** - 文本生成图表

### 核心特性
- ✨ 异步任务处理（不阻塞主线程）
- ✨ WebSocket 实时通信（双向数据推送）
- ✨ Mermaid 图表渲染（文本生成图表）
- ✨ 响应式设计（支持多设备）
- ✨ 模块化代码（易于维护和扩展）

## 🔄 CLI 依然可用

Web 界面是一个**补充功能**，不会影响现有的 CLI 工具。

你仍然可以使用命令行进行分析：
```bash
python -m src.cli analyze script.json --export report.md
```

两种方式互不影响，根据需求选择即可！
